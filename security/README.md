# Безопасность

### SSL Pinning
SSL Pinning — это техника, которая делает MITM-атаки практически невозможными.

Обычно Android доверяет _всем_ корневым сертификатам, которые есть в системе.  
Это значит:

- если злоумышленник подменит сертификат через поддельный прокси/VPN,  
- или установит свой корневой сертификат в систему,

то обычное HTTPS-соединение может быть перехвачено.

SSL Pinning ограничивает доверие к одному конкретному сертификату или публичному ключу.

То есть приложение _не доверяет_ системному хранилищу сертификатов, а доверяет конкретному сертификату сервера или публичному ключу.
