# Базы данных

### Нужны ли пермишены для работы с БД?

Нет. Не нужны.

SQLite/Room хранятся в приватной директории приложения:
```
/data/data/<package_name>/databases/
```
Эта директория недоступна другим приложениям, и доступ к ней есть только у твоего приложения.
- READ/WRITE внешнего хранилища не требуется.
- даже если пользователь не даёт доступ к Storage — база работает.

### Что происходит, если недостаточно места?

Когда система не может выделить место под запись в БД:

SQLite выбросит ошибку:
- SQLiteFullException, SQLiteDiskIOException
- Room превратит её в SQLiteConstraintException / SQLiteException
  
Что делает Android:
- может попытаться очистить кэш приложения
- может вызвать onTrimMemory()
- может убить процесс
- может очищать кеши других приложений

Но если места всё равно нет — запись НЕ произойдётся.

Как правильно обрабатывать эту ситуацию
- Ловить исключение в DAO/Repository
- Показывать пользователю: “Недостаточно места”
- Предлагать очистить место
- Минимизировать пишущие транзакции
- Сжимать БД периодически (VACUUM)
