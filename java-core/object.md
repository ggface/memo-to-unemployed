### Принципы equals

**Рефлексивность** — x.equals(x) возвращает true

**Симметричность** — x.equals(y) <=> y.equals(x)

**Транзитивность** — x.equals(y) <=> y.equals(z) <=> x.equals(z)

**Согласованность** — повторный вызов x.equals(y) должен возвращать значение предыдущего вызова, если сравниваемые поля не изменялись

**Сравнение с null** — x.equals(null) возвращает false

### Hashcode
Если два объекта равны по equals(), они должны иметь одинаковый hashCode()

Обратное не обязательно: объекты с одинаковым hashCode() могут быть не равны по equals()

hashCode() должен быть консистентным: для одного объекта должен возвращать одно значение, пока состояние не изменилось

### Клонирование
Копирует только сам объект, но не вложенные объекты

### Concurrency
**wait** свобождает монитор и ждет notify/notifyAll

**notify** Пробуждает один случайный поток из числа ожидающих на мониторе этого объекта

**notifyAll** Пробуждает все потоки, ожидающие на мониторе этого объекта.
Все пробужденные потоки конкурируют за захват монитора.
Только один поток сможет захватить монитор, остальные снова перейдут в режим ожидания.
