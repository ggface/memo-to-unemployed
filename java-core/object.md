# Object

## 📌 Что это

`Object` — базовый класс для всех классов в Java.

## 🔑 Основные методы

- `equals()`
- `hashCode()`
- `toString()`
- `clone()`
- `finalize()`

## ⚠️ Важно

- equals и hashCode должны быть согласованы
- нарушение контракта ломает коллекции
